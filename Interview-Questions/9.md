# Event Propagation

- If I have multiple events then the workflow of this event is called event propagation
- It could be
  - Top to bottom
  - Bottom to top
  - Randomly

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <link rel="stylesheet" href="src/style.css" />
  </head>
  <body>
    <main id="parent">
      <div id="child">
        <button id="button">Click Here</button>
      </div>
    </main>

    <script src="src/script.js"></script>
  </body>
</html>
```

## Button To Top / Event Bubbling

- JS file:

```js
const parent = document.querySelector('#parent');
const child = document.querySelector('#child');
const button = document.querySelector('#button');

parent.addEventListener('click', listener);

child.addEventListener('click', listener);

button.addEventListener('click', listener);

function listener(event) {
  console.log(event.currentTarget.tagName);

  // console.log(this.tagName); // Work same
}
```

- Output:

```txt
BUTTON
DIV
MAIN
```

## Top to Buttom / Event Capturing / Trickling

- Enable the 3rd parameter of event listener

```js
const parent = document.querySelector('#parent');
const child = document.querySelector('#child');
const button = document.querySelector('#button');

parent.addEventListener('click', listener, true);

child.addEventListener('click', listener, true);

button.addEventListener('click', listener, true);

function listener(event) {
  console.log(event.currentTarget.tagName);

  // console.log(this.tagName); // Work same
}
```

- Output:

```txt
MAIN
DIV
BUTTON
```

## Random Flow

- We would like to want a flow
- DIV > BUTTON > MAIN

```js
const parent = document.querySelector('#parent');
const child = document.querySelector('#child');
const button = document.querySelector('#button');

parent.addEventListener('click', listener);

// Enable 3rd parameter first
child.addEventListener('click', listener, true);

button.addEventListener('click', listener);

function listener(event) {
  console.log(event.currentTarget.tagName);

  // console.log(this.tagName); // Work same
}
```

- Output:

```txt
DIV
BUTTON
MAIN
```
