# Function Constructor

- Create a base blueprint & create n number of objects
- Using **Function Constructor**, I can create base blueprint
- Normal function - naming - _camelCasing_
- Constructor function - naming - _PascalCasing_

```js
function Person(fname, lname, contact) {
  this.fname = fname;
  this.lname = lname;
  this.contact = contact;
  this.getName = function () {
    console.log(this.fname + ' ' + this.lname);
  };
}

const person1 = new Person('Rezoan', 'Shakil', '+9999');
console.log(person1);

const person2 = new Person('Shakib', 'Al-hasan', '+8888');
console.log(person2);
```

# ES6 Class

- After ES6, instead of using **Function Constructor**, inspired to use **Class**

```js
class Person {
  constructor(fname, lname, contact) {
    this.fname = fname;
    this.lname = lname;
    this.contact = contact;
  }

  getName = function () {
    console.log(this.fname + ' ' + this.lname);
  };
}

const person1 = new Person('Rezoan', 'Shakil', '9999');
console.log(person1);

const person2 = new Person('Shakib', 'Al-hasan', '8888');
console.log(person2);
```
